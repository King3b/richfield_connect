# Richfield Connect 🎓💬

Richfield Connect is a front-end web application prototype developed for the WEB TECHNOLOGY 511 assignment.

The platform is designed as an academic social networking system where students can:

- Create academic profiles
- Share posts with other students
- Interact through likes and feed updates
- Build collaborative academic connections

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Browser Local Storage

---

## 📂 Pages Included

- Home Page
- About Page
- Sign-Up Page
- Profile Page
- Feed Page

---

## ✨ Features

### ✅ User Registration

- Form validation using JavaScript
- Inline error messages
- Password validation
- Email format validation
- Student number validation

### ✅ Dynamic Profile System

- User data stored in localStorage
- Automatic profile population
- Live profile preview using jQuery

### ✅ Interactive Feed

- Create posts dynamically
- Like button functionality
- Delete posts with confirmation
- Date and time stamps

### ✅ Data Persistence

- Posts saved in localStorage
- Data remains after page refresh

### ✅ jQuery UI Enhancements

- Fade-in animations
- Hover effects
- Slide toggles
- Interactive transitions

---

## 📱 Responsive Design

The website is fully responsive and adapts to:
